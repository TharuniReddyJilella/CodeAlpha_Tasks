## Hangman Game

# Overview

The Hangman Game is a console-based word guessing game developed in Python. Players attempt to discover a hidden word by guessing one letter at a time within a limited number of attempts.
The game includes multiple categories, score tracking, and input validation, providing an engaging way to practice logical thinking and problem-solving.

# Features

- Multiple word categories
  - Kitchen Tools
  - Birds
  - Fruits
  - Animals
- Random word selection
- Letter-by-letter guessing
- Duplicate guess detection
- Input validation
- Score tracking
- Limited incorrect attempts
- Automatic category rotation


# Technologies Used

- Python 3
- Random Module
- Loops
- Conditional Statements
- Lists
- String Manipulation
