## StockPortfolio Tracker

# Overview

The Stock Portfolio Tracker is a console-based Python application that calculates the total investment value of a stock portfolio based on user-selected stocks and their quantities.
The application allows users to select stocks from a predefined dataset, calculates individual investment values, and displays the overall portfolio value.


# Features

- Displays available stocks and prices
- Accepts multiple stock entries
- Calculates investment value for each stock
- Generates total portfolio value
- Handles invalid stock symbols
- Simple command-line interface


# Technologies Used

- Python 3
- Dictionaries
- Loops
- Conditional Statements
- User Input Handling
