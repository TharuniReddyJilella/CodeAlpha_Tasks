## Smart Chatbot

# Overview

The Rule-Based Smart Chatbot is a console-based Python application that simulates a simple conversational assistant using predefined rules. It interacts with users through text-based conversations, responds to common queries, and provides recommendations and informational responses across multiple topics.

The project demonstrates how conditional logic can be used to build an interactive chatbot without relying on machine learning or external APIs.

# Features

- Interactive text-based conversation
- Personalized interaction using the user's name
- Mood-based responses
- Answers questions related to:
  - Python
  - Java
  - Data Structures and Algorithms (DSA)
  - Database Management Systems (DBMS)
- Movie recommendations
- Book recommendations
- Programming jokes
- Motivational quotes
- Continuous conversation until the user exits
- Graceful conversation termination


# Technologies Used

- Python 3
- Functions
- Conditional Statements
- Loops
- String Manipulation
