# CodeAlpha Python Internship Tasks

This repository contains the Python projects that I have completed during my CodeAlpha Python Programming Internship. The purpose of these tasks is to improve my understanding of Python programming by building simple console-based applications using basic programming concepts.

These projects helped me to practice problem-solving, logical thinking, and writing structured Python programs.


## Task 1 - Hangman Game

# Description

The Hangman Game is a console-based word guessing game where the player has to guess a hidden word one letter at a time. The game contains multiple categories such as Kitchen Tools, Birds, Fruits, and Animals. A random word is selected from the chosen category, and the player gets six chances to guess the word correctly.

# Features

* Multiple word categories
* Random word selection
* Score tracking
* Six incorrect attempts
* Replay option
* Input validation

# Concepts Used

* Random Module
* Lists
* Strings
* While Loop
* If-Else Statement


## Task 2 - Smart Chatbot

# Description

The Smart Chatbot is a simple rule-based chatbot developed using Python. It interacts with the user by responding to predefined questions and provides information related to programming topics, books, movies, jokes, and motivational quotes.

# Features

* Greets the user
* Personalized conversation
* Programming-related information
* Movie and book suggestions
* Motivational quotes
* Simple jokes
* Ends conversation using "Bye"

# Concepts Used

* Functions
* Loops
* If-Elif Statements
* String Handling
* User Input


## Task 3 - Stock Portfolio Tracker

# Description

The Stock Portfolio Tracker is a console-based Python application that allows users to calculate the total investment value of their stock portfolio using predefined stock prices. The program validates user input to prevent invalid entries and also provides an option to save the portfolio summary to a text file.

# Features

* Displays available stocks and their prices
* Accepts stock name and quantity from user
* Calculates investment value
* Displays portfolio summary


# Concepts Used

* Dictionary
* Loops
* Arithmetic Operations
* Input and Output
